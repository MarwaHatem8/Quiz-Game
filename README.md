# 🎮 Quiz Game

An interactive quiz game built with **HTML**, **CSS**, and **JavaScript**. The application supports both **Arabic** and **English**, providing an engaging learning experience through multiple-choice questions, instant feedback, and score tracking.

---

## 🌐 Live Demo

**Play the game:**
https://marwahatem8.github.io/Quiz-Game/

---

## 📌 Features

* 🌍 Arabic and English language support
* ❓ Multiple-choice questions
* 🎵 Sound effects for user interaction
* 📊 Real-time score calculation
* 📱 Responsive design for different screen sizes
* 🎨 Clean and user-friendly interface

---

## 🛠 Built With

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/MarwaHatem8/Quiz-Game.git
```

Open the project folder:

```bash
cd Quiz-Game
```

Finally, open **index.html** in your browser.

---

## 🎮 How to Play

1. Open the game.
2. Choose your preferred language.
3. Read each question carefully.
4. Select the correct answer.
5. Continue until all questions are completed.
6. View your final score.
